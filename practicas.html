Práctica 1

Introducción

Se desarrollarán dos códigos, usando dos Arduinos, el primero se encargará de enviar una cantidad determinada de pulsos con un tiempo dado como parámetro. El segundo recibirá los pulsos y se encargará de contarlos.Además, se usará un led para demostrar el envío de pulsos.

Objetivo General

Se enviarán pulsos de un arduino a otro y podremos de ver la cantidad de pulsos enviados y recibidos de uno a otro

Marco teórico

Arduino es una plataforma de creación de electrónica de código abierto, la cual está basada en hardware y software libre, flexible y fácil de utilizar para los creadores y desarrolladores. Esta plataforma permite crear diferentes tipos de microordenadores de una sola placa a los que la comunidad de creadores puede darles diferentes tipos de uso.

Definición recuparada de: https://www.xataka.com/basics/que-arduino-como-funciona-que-puedes-hacer-uno

Materiales usados

• Dos Arduinos
• Un Protoboard
• Botones
• LED’s

Desarrollo

Se colocará un led en la salida de nuestro Arduino, el cual, enviará una señal para que se puedan observar de una mejor forma, los Arduinos se van a conectar usando los puertos digitales, Utilizaremos cables para realizar las conexiones para que permitan una comunicación.

Código que manda pulsos

void setup{
pinMode(10,HIGH); //Se inicializan los pines de salidas
Pulsos(5,1000); // Llamado a la función
}
void Pulsos(int cantidad, int duracion){ //función para crear los pulsos
int i = 0;
while(i <= cantidad) { //manda pulsos determinados por la variable cantidad
digitalWrite(10,HIGH); //Envía pulsos por el pin 10
delay(duracion/2); // se divide el tiempo establecido en el llamado de la función y se mantiene encendido la mitad del tiempo indicado
digitalWrite(10,LOW); //Apagado
delay(duracion/2);
i++;
}
}
void loop() {}

Código que recibe pulsos

const int pin1 = 9; //El pin 9 es el receptor
int cont = 0; //contador
void setup() {
pinMode(pin1, INPUT);
Serial.begin(9600); //Establece la comunicación
}
void loop(){
if(digitalRead(pin1) == HIGH){ //se hace un if con la finalidad de que detecte un inicio (HIGH)
cont++; //incrementa el contador
while(digitalRead(pin1) == HIGH);
Serial.println(cont); //imprime los pulsos contados
}
